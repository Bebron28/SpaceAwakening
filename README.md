# SpaceAwakening

Небольшой 2D шутер/аркада, где игрок управляя космическим кораблем должен уничтожать инопланетные корабли.

## Управление
- A/D / стрелки - движение
- ЛКМ / W / Стрелка вверх - атака

## Технологии
- Unity 6.0
- C#

## Структура проекта
- Assets/Prefabs - готовые обьекты(префабы)
- Assets/Scenes - игровые сцены
- Assets/Scripts - все скрипты

## Как запустить
1. скачать репозиторий
2. Открыть в Unity Hub
3. Открыть сцену Assets/Scenes/Menu.unity
4. Нажать Play

## Что реализовано 
- Движение персонажа
- Атака
- Спавн врагов в случайном месте
- Система ХП
- Система уровней
